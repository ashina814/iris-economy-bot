# アイリス経済Bot 利用ガイド

この文書は、アイリス経済Botを実際に使う住民向けの仕様まとめです。READMEよりも利用者目線で、何を押せばよいか、Risをどう稼ぐか、何に使うか、称号がどう上がるかを中心にまとめています。

最初に覚える入口は、基本的に次の3つです。

```text
/アイリス  ... ホーム。財布、ログボ、カード、ショップ、貢献への入口
/カード    ... 自分のプロフィールカードを表示
/ショップ ... 買う、見る、入札する、持ち物を見る
```

売りたい人は `/自分の店`、運営は `/管理` を使います。

## 1. 最初にやること

Botを使い始める人は、まず `/アイリス` を開いて `参加` を押します。コマンドで使う場合は `join` です。

参加すると、初期資本として **100,000 Ris** を受け取ります。これは貸付ではなく、アイリス中央台帳からの初期配布です。返済は不要です。

招待経由で入った場合は、招待した人にも招待報酬が入ります。招待された人にも追加で **250 Ris** が入ります。招待報酬は1日の有償上限があり、無限に稼げるわけではありません。

参加後は、まず `/カード` で自分の状態を確認し、`ログボ` を受け取り、ショップや貢献台帳を見ていくのが基本です。

## 2. `/アイリス` とは

`/アイリス` はBotのホーム画面です。財布、ランク、ログボ、カード、ショップ、貢献、通知設定への入口をまとめたパネルです。

ホームでは主に次の操作ができます。

```text
参加       ... 初期資本を受け取って経済圏に入る
カード     ... プロフィールカードを見る
ログボ     ... 24時間ごとのログイン報酬を受け取る
ショップ ... 商品購入、民営ショップ、公式オークションへ進む
自分の店   ... 自分のショップ管理へ進む
貢献       ... 招待、Bump/Up、ランキングを見る
通知設定   ... DM通知のON/OFF
```

通話報酬は現在、自動精算です。通話ラウンジや手動VC精算の入口は廃止されています。VCに入ると在室時間が記録され、退出・移動時、または在室中の定期処理で自動反映されます。

## 3. `/カード` とは

`/カード` は、自分のプロフィールカードを表示する機能です。単なる残高表示ではなく、アイリス内での現在位置をまとめて見られるカードです。

カードに出る主な情報は次の通りです。

```text
財布          ... 今すぐ使えるRis
純資産        ... 財布などを含めた総合的な資産評価
経済ランク    ... 純資産に応じた経済称号
TCランク      ... テキストチャット活動による称号
VCランク      ... 通話活動による称号
現在の肩書き  ... TC/VCの到達状況などから決まる表示称号
カードスタイル ... 活動量や所持品に応じた見た目
```

公式ショップの `黒金カード枠` を持っていると、プロフィールカードの見た目に反映されます。

カードは、今の残高だけでなく「どれくらい話しているか」「どれくらい通話にいるか」「どの称号に近いか」を見る場所です。

## 4. Risとは

`Ris` はアイリス経済Botの通貨です。発行元は `アイリス中央台帳` です。

Risは主に次の用途で使います。

```text
公式ショップで商品を買う
民営ショップでユーザー出品の商品を買う
公式オークションに入札する
二人宿のシークレット宿や人数追加に使う
ユーザー間で送金する
ロール、称号、権利、サービスなどの取引に使う
```

Risはサーバー内の遊び、取引、称号、サービスを回すための通貨です。ためるだけでなく、ショップや宿で使う前提の設計です。

## 5. Risの稼ぎ方

Risは複数の経路で増えます。

### 5.1 初期資本

`join` または `/アイリス` の `参加` で **100,000 Ris** を受け取ります。新規住民が最初からショップや宿を使えるようにするための初期通貨です。

### 5.2 ログボ

`daily` または `/アイリス` の `ログボ` で、24時間ごとの報酬を受け取れます。

公式ショップの `常連カード` を持っていると、ログボに **+250 Ris** が上乗せされます。

### 5.3 労働

`work` で短いクールダウンの労働報酬を受け取れます。労働内容はランダムなテキストで表示され、報酬も内容によって変わります。

公式ショップの `深座りの椅子` を持っていると、労働報酬が **+15%** されます。

### 5.4 給付金

`subsidy` または `給付金` で、一定間隔の給付金チャレンジができます。ログボや労働とは別の小さな収入源です。

### 5.5 テキストチャット活動

通常チャットで発言すると、TC XPと少量のRisが入ります。標準設定では、一定クールダウンごとに有効発言として処理されます。

標準値は次の通りです。

```text
TC XP: 1〜2 XP / 有効発言
TC Ris: 1〜4 Ris / 有効発言
TCクールダウン: 60秒
```

この値は運営パネルの `XP設定` から変更できます。

### 5.6 VC活動

VCに入っている時間でVC XPが増えます。VC報酬は手動精算ではなく、自動精算です。

標準値は次の通りです。

```text
VC XP: 1 XP / 分
1回の精算上限: 240分
```

RisとしてのVC報酬もあります。もともとの報酬設計では、VC滞在1分あたりのRis報酬があり、VCレベルに応じて伸びます。現在は退出・移動時と在室中の定期処理で反映されます。

### 5.7 招待

誰かを招待し、その人が参加して初期資本を受け取ると招待成立です。招待した人は、招待階級に応じた報酬を受け取ります。招待された人にも **250 Ris** が入ります。

招待には1日の有償上限があります。成立数は称号とランキングにも反映されます。

### 5.8 Bump/Up

DISBOARDの `/bump` または `/up` が成功すると、Bump/Up貢献として同じカウントに加算されます。

Bump/Up回数が増えると階級が上がり、1回あたりの報酬も増えます。

### 5.9 給与配布

運営は `/管理` の `残高操作` から、ロール保持者に一括でRisを配布できます。これは中央台帳からの発行であり、実行した管理者の財布から引かれるわけではありません。

## 6. Risの使い道

### 6.1 公式ショップ

公式ショップの商品は運営がショップ管理パネルから追加・編集します。購入前には確認画面が出ます。所持上限、在庫、販売開始・終了日時を商品ごとに設定できます。

ロール付き商品は、購入時にBotがロール付与を試みます。期限日数を設定した場合は期限後に回収します。Botの権限やロール階層の都合で付与できない場合でも購入記録は消えず、運営の公式商品対応キューから再試行または手動対応できます。

当面は、商品効果は説明文として表示される所持型アイテムです。労働ボーナスなどBot処理を伴う効果は個別に実装されるまで自動では発動しません。

### 6.2 IRIS商店街（民営ショップ）

ユーザー同士のサービス売買の市場です。店舗の開設操作は不要で、初出品すると販売者ページ（あなたの店）が自動で作られます。1人1店舗です。

買う側は `/ショップ` から入り、「店舗を見る（人から探す）」か「商品を探す（目的から探す）」で商品にたどり着きます。店舗を開くと、その販売者の全出品が見られます。

出品時に選ぶのはカテゴリだけです（通話 / ゲーム / 制作 / 相談 / ネタ / その他。検索用の分類で、処理はどれも同じです）。

```text
出品に入力するもの: 商品・サービス名 / 説明 / 価格 / 受付上限（空欄で無制限）
```

取引はすべてエスクロー方式です。

```text
購入（依頼内容を添えられる）
→ 代金は取引完了までIRISが預かる
→ 販売者が受注 → 対応 → 対応完了
→ 購入者が受取確認（または72時間で自動完了。24時間前に通知）
→ 手数料差引後のRisを販売者へ支払い
```

- 販売者が受注する前なら購入者はキャンセルできます。販売者は辞退して全額返金できます
- 問題があれば「問題を報告」で運営対応待ちにできます（報告中の取引は自動完了しません）
- 進行状況は「自分の取引」からいつでも確認・操作できます
- 販売者がサーバーを退出すると、その出品は新規購入停止になります（進行中の取引は残ります）

商店街Forumが設定されているサーバーでは、販売者ごとに店舗のForum投稿ができ、そこに出品がまとまって表示されます。店舗投稿の商品一覧から購入に進むこともできます。Forumがなくても `/ショップ` と `/自分の店` だけですべて完結します。

審査はリスク型です。初回出品・高額出品・要確認状態の販売者・通報停止後の再提出だけが運営確認後の公開になり、それ以外は即公開されます。

旧仕様（商品タイプ「称号/アイテム/チケット/権利/サービス/セット商品」、販売方式「買い切り/期間制」、店を開く操作、一律審査）は廃止済みのlegacy仕様です。旧仕様で購入・出品した記録はそのまま閲覧でき、旧注文は「自分の取引」から確認できます。

### 6.3 公式オークション

公式オークションは運営が作成します。ユーザーは入札額を入力して参加します。

主な仕様は次の通りです。

```text
開始価格あり
必要に応じて即決価格あり
最低入札額あり
入札額は一時拘束
上書きされた人には自動返金
終了直前の入札で自動延長
終了時に最高入札者へ商品付与
```

### 6.4 二人宿

二人宿はVC作成機能です。運営がパネルを設置して使います。

```text
公開宿          無料 / 誰でも見える / 基本2人
シークレット宿  10,000 Ris / 選んだ相手と自分だけ見える
人数追加        追加1人につき5,000 Ris
期限            12時間で自動終了
制限            1人につき作成できる宿は1つまで
```

空になった宿VCは自動削除されます。

### 6.5 送金

`/送金` で他の住民へRisを送れます。Botには送金できません。

## 7. 称号とランクの全体像

アイリスには複数の称号軸があります。

```text
TCランク      ... テキストチャット活動
VCランク      ... 通話活動
経済ランク    ... 純資産
招待階級      ... 招待成立数
Bump/Up階級   ... Bump/Up成功回数
```

特に重要なのはTCランクとVCランクです。どちらも21段階あり、最上位の `アイリス` は隠し称号です。

## 10. 経済ランク

経済ランクは純資産で決まる称号です。

| 称号 | 必要純資産 |
|---|---:|
| 未上場 | 0 Ris |
| 個人商店 | 1,000 Ris |
| 流動性提供者 | 7,000 Ris |
| 地区銀行 | 25,000 Ris |
| 経済圏中枢 | 80,000 Ris |
| アイリス財閥 | 200,000 Ris |

初期資本が100,000 Risなので、参加直後から経済ランクは高めに見えます。これは、新規住民がすぐショップを動かせるようにするためです。

## 11. 招待階級

招待階級は、成立した招待人数で上がります。

| 階級 | 必要成立数 | 成立報酬 |
|---|---:|---:|
| 勧誘見習い | 0人 | 900 Ris |
| 声かけ屋 | 3人 | 1,300 Ris |
| 人脈師 | 10人 | 1,900 Ris |
| 門番長 | 20人 | 2,700 Ris |
| アイリスの広告塔 | 35人 | 3,800 Ris |
| 伝説の勧誘師 | 50人 | 5,000 Ris |

招待された人が参加して初期資本を受け取ると成立します。招待された人にも250 Risが入ります。

## 12. Bump/Up階級

Bump/Up階級は、DISBOARDの `/bump` または `/up` の成功回数で上がります。内部的には同じ貢献として扱われます。

| 階級 | 必要回数 | 1回の報酬 |
|---|---:|---:|
| はじめての宣伝 | 0回 | 500 Ris |
| 常連宣伝員 | 10回 | 700 Ris |
| 広報担当 | 30回 | 900 Ris |
| 宣伝部長 | 60回 | 1,200 Ris |
| アイリスの拡声器 | 100回 | 1,500 Ris |

ランキング上もBump/Upはまとめて扱われます。

## 13. ランキング

ランキングはランク確認パネル、またはコマンドで確認できます。

```text
rank net     ... 純資産ランキング
rank text    ... TCランキング
rank vc      ... VCランキング
rank invite  ... 招待ランキング
rank bump    ... Bump/Upランキング
```

ランキングは現在サーバーにいるメンバーを対象にします。抜けたユーザーはランキングから除外されます。表示はDiscordメンションとサーバーニックネームを優先するため、名前被りがあっても誰か分かるようになっています。

ランク確認パネルから押したランキングは、自分だけに見える非公開表示です。自分がTop10外でも、自分の順位とTop10までの差が表示されます。

VCランキングは称号とXPを表示します。分数は表示しません。

## 14. 通知と昇格

TC、VC、招待、Bump/Upなどで階級が上がると昇格通知が出ます。通知先は運営パネルのランク設定から指定できます。

VC昇格通知は、通話分数、獲得XP、獲得Risなどの細かい内訳を出さず、誰がどの称号になったかを中心に表示します。

個人向けの通知は、通知設定からDM通知のON/OFFを切り替えられます。

## 15. 運営パネルでできること

`/管理` は運営用です。主に次の操作があります。

```text
ショップ管理   ... 公式商品の追加・編集・停止、在庫・販売期間、ロール、対応キュー、審査、取引対応、公式オークション
残高操作         ... 個人セット、加算、減算、ロール一括セット、給与配布
ランク設定       ... 昇格通知先、ランク確認パネル送信、XP設定
常設パネル送信   ... 住民向けパネルをチャンネルに設置
```

XP設定では、TC/VCの伸び方を調整できます。

```text
軽め   TC 2〜3XP / 45秒、VC 2XP/分
標準   TC 1〜2XP / 60秒、VC 1XP/分
重め   TC 1XP / 90秒、VC 1XP/分
```

調整できる項目は次の通りです。

```text
TC XP下限
TC XP上限
TCクールダウン
VC XP/分
```

## 16. よくある流れ

新しく入った人は、まず次の流れで使えます。

```text
1. /アイリス を開く
2. 参加を押して 100,000 Ris を受け取る
3. /カード で自分の状態を見る
4. ログボを受け取る
5. /ショップ で公式ショップや民営ショップを見る
6. 通常チャットやVCでTC/VCランクを育てる
7. 招待やBump/Upで貢献階級を育てる
```

お金を増やしたい人は、ログボ、労働、給付金、VC、招待、Bump/Up、販売を使います。

お金を使いたい人は、公式ショップ、民営ショップ、公式オークション、二人宿、送金を使います。

称号を育てたい人は、TC、VC、招待、Bump/Upを意識すると分かりやすいです。
